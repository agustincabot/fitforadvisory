Método

Tu idea es correcta: primero entender la empresa, luego diseñar algo a medida. Acá lo estructuro en un proceso claro (tipo consultora), pero ejecutable.

1 Diagnóstico (rápido y serio)

2–3 reuniones para entender el negocio, números, problemas y restricciones. Se define el “driver” principal.

  • ✓
    Mapa económico del negocio Ingresos, costos, mix, precios, descuentos, overhead, CAPEX y working capital.
  • ✓
    Hipótesis y quick wins Acciones de impacto rápido vs iniciativas estructurales.
  • ✓
    Plan de trabajo Alcance, entregables, cronograma y responsables.
2 Diseño a medida

Construimos el caso: modelo financiero, escenarios y decisiones recomendadas (no “plantillas”).

  • ✓
    Escenarios y sensibilidad Qué pasa si cambian precios, volumen, costos, tipo de cambio, plazos de cobro/pago.
  • ✓
    Decisiones y roadmap Prioridades claras: qué hacer primero, qué medir y cómo sostenerlo.
  • ✓
    Presentación ejecutiva Resumen para dueños/directorio: claro, defendible y accionable.
3 Implementación (opcional)

Si querés, acompaño la ejecución: seguimiento mensual, control de desvíos, decisiones y ajuste del plan.

output = diagnóstico + plan + escenarios + tablero / modelo (según necesidad)
⟠ Equipo extendido

El foco es el análisis económico-financiero. Para ejecución integral, sumo especialistas por proyecto: legal / impuestos, marketing, operaciones, sistemas, RR.HH. (según necesidad).

Ventaja: seniority + agilidad, sin costos fijos de estructura pesada.

A medida. Primero entiendo el negocio y el problema. Luego diseño el modelo, los escenarios y el plan de decisiones. No vendo un Excel genérico.

¿Qué entregables puedo esperar?

Depende del caso: diagnóstico + roadmap + modelo financiero con escenarios / sensibilidad, y si aplica un tablero o set de reportes ejecutivos.

¿Sos vos solo o hay equipo?

Soy boutique senior: yo lidero y ejecuto el núcleo económico-financiero. Para necesidades específicas, se suma una red de especialistas por proyecto (legal/impuestos/operaciones/sistemas/marketing).
